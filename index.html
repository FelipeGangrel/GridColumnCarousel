Lets scroll some columns

Welcome to the fantastic world of scrollable collumns for your favorite responsive grid system! GridColumnCarousel lets you scroll columns that scale up and down with your responsive grid. When your grid system hits breakpoints and changes the amount of columns you have pr page, GridColumnCarousel will automatically change the number of columns per slide.

The script is implemented and maintained by anysom

But how do I do it?!!?

Do not stress my friend, it is as simple as ****** your own ***...

First build the markup for your slider. There are 2 key classes you must use. The 'grid-column-carousel' defines the top level element of your carousel, and the 'grid-column-carousel__list' defines the ul containing the list items.

The class 'grid-column-carousel__page-indicators' is the default element to contain the slide indicators. But indicators are optional, and you can supply your own selector for defining the element to contain them.

Please note that in the example below, I use some Bootstrap specific classes, but these could be whatever grid you want.

  
    <div class="grid-column-carousel row">
      <ul class="grid-column-carousel__list">
        <li class="col-xs-12 col-sm-6 col-md-4 col-lg-3">
          <h3>1 This is a column with a lot of text in</h3>
        </li>
        <li class="col-xs-12 col-sm-6 col-md-4 col-lg-3">
          <h3>2 This is a column with a lot of text in</h3>
        </li>
        <li class="col-xs-12 col-sm-6 col-md-4 col-lg-3">
          <h3>3 This is a column with a lot of text in</h3>
        </li>
        <li class="col-xs-12 col-sm-6 col-md-4 col-lg-3">
          <h3>4 This is a column with a lot of text in</h3>
        </li>
        <li class="col-xs-12 col-sm-6 col-md-4 col-lg-3">
          <h3>5 This is a column with a lot of text in</h3>
        </li>
        <li class="col-xs-12 col-sm-6 col-md-4 col-lg-3">
          <h3>6 This is a column with a lot of text in</h3>
        </li>
      </ul>
      <div class="grid-column-carousel__page-indicators">
        <ul>
            <li class="dot active"></li>
        </ul>
      </div>
    </div>

Then include the script, and initialize the GCCarousel with your options. As a minimum you must supply a javascript object representing your slider, and the class list that defines the grid on your items in the slider.

  
    <!-- Include the script--<
    <script src="js/GridColumnCarousel.js"></script>
    
    <!-- initialize the carousel -->
    <script>
      window.onload = function() {
        var options = {
          elem: document.getElementsByClassName('grid-column-slider')[0],
          gridColClasses: 'col-xs-12 col-sm-6 col-md-4 col-lg-3'
        };
         
        var gCCarousel = new GCCarousel(options);      
      };
    </script>

Works with your favorite grid system

GridColumnCarousel is grid system agnositc, and therefore you can use it with whichever grid system you like.

  • Bootstrap

  • Foundation

No independence upon third party libraries

The GridColumnCarousel is written in pure javascript, and as such does not rely on jQuery, Underscore, Lodash or other libraries..
